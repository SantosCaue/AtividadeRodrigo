Microsoft

Fundada em 4 de abril de 1975 por Bill Gates e Paul Allen, dois amigos universitários, a empresa começou como uma pequena startup que desenvolvia e vendia interpretadores BASIC para computadores pessoais. Em 1980 a IBM, na época lider no mercado de computadores procurou a Microsoft para desenvolver um Sistema Operacional para o computador deles. Em uma reunião, Bill Gates disse que tinha um sistema e que poderia vende-lo, ele o vendeu por US$ 8 milhões, porém, Gates não tinha o sistema pronto. Gates e Allen conheciam um homem que havia criado um sistema operacional primitivo, mas que serviu de base para criação do MS-DOS. Gates e Allen compraram o sistema por US$50 mil, o homem que havia criado ficou abismado e vendeu na hora, ele pensava que o que ele tinha feito nao valia nada e era só um "programinha". Em 1985 a Microsoft lançou o Windows, um sistema operacional com interface gráfica que rapidamente se tornou o mais popular do mundo. O Windows foi responsável por "jogar" a Microsoft para o sucesso, tornando-a uma das maiories empresas do mundo.

Alguns dos principais marcos da história da Microsoft
  • 1975: Bill Gates e Paul Allen fundam a Microsoft
  • 1980: A Microsoft desenvolve o MS-DOS para a IBM
  • 1985: A Microsoft lança o Windows
  • 1990: A Microsoft lança o pacote de aplicativos Office
  • 1995: A Microsoft lança o Internet Explorer
  • 2001: A Microsoft lança o console de games Xbox
  • 2010: A Microsoft lança a plataforma de nuvem Azure

A Microsoft sempre foi uma empresa que focou em softwares para os computadores, mas isso não quer dizer que ela não produz hardware. A Microsoft possui algumas linhas de notebooks extremamente portáteis, tablets e até celulares.
A Microsoft é a maior empresa de desenvolvimento de software e computação em nuvem do mundo. O valor de mercado dela é de US$ 2,62 trilhões
